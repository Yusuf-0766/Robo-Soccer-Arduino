# Robo Soccer Arduino Project

This project was developed as part of a team for a Robo-Soccer competition using Arduino.

## Overview
We built a robot that can be controlled wirelessly to move around the field and play Robo-Soccer. The system uses DC motors for movement and a Bluetooth module for control.

## Features
- Wireless control using a Bluetooth module  
- Basic movement control (forward, backward, turning)  
- Arduino-based system with motor driver integration  
- Team-based development and testing  

## Technologies Used
- Arduino  
- Embedded C/C++  
- Motor Driver (L298N)  
- Bluetooth module
